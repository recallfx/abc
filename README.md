# Address book challenge SPA

## Global dependencies

```
$ npm install -g gulp-cli casperjs phantomjs-prebuilt
```

## Setup dependencies

````
$ npm install
$ bower install --force-latest
```

### Server

Development: `$ gulp serve-dev`, production: `$ gulp serve-dist`.

## Tests

```
$ npm test
```
